# **App Name**: StudyAI DZ

## Core Features:

- Google Sign-In with Email Verification: Allow users to sign in with their Google accounts and verify their email addresses.
- Chatbot: Implement a chatbot that answers student questions using Firebase AI Logic (Gemini) and stores the answers in Firestore.
- PDF Summarization: Allow students to upload PDF documents, summarize the text using Firebase AI Logic, and save the summaries in Firestore.
- AI-Powered Mind Map Generation: Convert text to a visual mind map tool using AI and render it with D3.js or Mermaid.js.
- AI Quiz Generator: Generate quizzes using Firebase AI Logic with multiple-choice questions and a timer. Store the results in Firestore.
- Community Forum: Implement a community forum where students can ask questions, post discussions, and upvote/downvote content.
- Admin Dashboard: Create an admin dashboard with statistics (active users, popular subjects, content counts) and moderation tools.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), evokes a sense of trust, calmness and openness to information, complementing the educational aspect of the platform.
- Background color: Off-white (#FAFAFA) provides a clean, neutral backdrop, preventing eye strain during prolonged study sessions.
- Accent color: Green (#90EE90) provides an element of freshness and subtly emphasizes growth, improvement and success, relating to the students journey using the application.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of text, 'PT Sans' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets. Note: currently only Google Fonts are supported.
- Use clear and simple icons for navigation and features, ensuring ease of use for students.
- Responsive layout design for both mobile and desktop, providing a seamless experience across devices.
